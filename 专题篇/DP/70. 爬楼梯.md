## 代码
```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 1: return 1 #因为下文对dp[2]赋值了，防止空指针所以小于一直接返回
        dp = [0] * (n+1)
        dp[1] = 1
        dp[2] = 2
        for i in range (3,n+1):
            dp[i] = dp[i-1] +dp[i-2]
        return dp[-1]
```
## 复杂度
- 时间复杂度：$O(n)$ 
- 空间复杂度：$O(n)$
